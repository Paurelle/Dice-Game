# Dice-Game
Version 1.0
Vous trouverez sur ce projet un site contenant un petit jeu de dés.

/* Règles */

Le jeu comprend 2 joueurs sur un seul et même écran. 
Chaque joueur possède un score temporaire et un score global.
À chaque tour, le joueur peut lancer un dé autant de fois qu'il le souhaite. 
Lors de son tour, le joueur peut décider à tout moment de:
- Cliquer sur l’option “Hold”, qui permet d’envoyer les points du tour vers les points globals. 
Ce sera alors le tour de l’autre joueur.
- Lancer le dé. S’il obtient un 1, son score du tour est perdu et c’est la fin de son tour.
Le premier joueur qui atteint les 100 points sur le score global gagne le jeu.
